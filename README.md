#  Student Grade Management System (SQL)

Цей проєкт — система управління базою даних студентів, розроблена для відстеження успішності, розподілу стипендій та аналітики по групах. 

Написано мовою **SQL** з використанням середовища **DBeaver**.

##  Технології
* **SQL** (PostgreSQL діалект)
* **DBeaver** (для роботи з БД)
* **Git** (контроль версій)
* **Python** (створення даних)

##  Функціонал
У цьому проєкті реалізовані такі можливості:
* **Створення структури БД:** Таблиця студентів з типами даних (`TEXT`, `INT`,`PRIMARY KEY`).
* **Наповнення даними:** Додавання тестових записів про студентів.
* **Фільтрація:** Показ студентів старших 30.
* **Сортування:** Формування списку студентів від молодіших до старших.

##  Структура бази даних

Таблиця `students` має такі поля:
* `id`: Унікальний ідентифікатор.
* `name`: Ім'я.
* `surname`: Прізвище.
* `age`: Вік студента по якому відбуваєтся сортування в queries.
